Introduction to SQL

SQL is a standard language for accessing and manipulating databases. Using SQL, you can insert records, update records and delete records. You can also create new database objects such as databases and tables.

More advanced features include creating stored procedures (self contained scripts), views (pre-made queries), and setting permissions on database objects (such as tables, stored procedures, and views).

So, what can SQL do?

  • SQL can execute queries against a database
  • SQL can retrieve data from a database
  • SQL can insert records in a database
  • SQL can update records in a database
  • SQL can delete records from a database
  • SQL can create new databases
  • SQL can create new tables in a database
  • SQL can create stored procedures in a database
  • SQL can create views in a database
  • SQL can set permissions on tables, procedures, and views

Common Uses

SQL is used to manage databases. Data Analysts will use SQL to query tables of data and derive insights from it. Data Scientists will use SQL to load data into their models. Data Engineers and database admnistrators will use SQL to ensure that everybody in their company has easy and intuitive access to the data they need.

Most websites will have a SQL backend that automatically stores everything from user details to user engagement across a particular app.

Creating and Droping a Database

To get started, we going to create a new database writing some SQL code, to this we are going to create a database that is going to contain some data about countries, so we called hello_world, so to create the database we write something like this: CREATE DATABASE hello_world; Creating a database does not select it for use; you must do that explicitly. To make hello_world the current database, you have to use this statement: USE hello_world; So, to drop a database, you have to use the next code: DROP DATABASE hello_world;

SELECT Basics

The most simple way to retrieve data from a table is with the next code: SELECT * FROM countries; This will retrieves everything from a table. This form is useful when you want to review your entire table, for example, after you've loaded it with your initial data set. In this case, this code will show a result like the table that we have seen before.

In SQL we can retrieve specifics amounts of data by a WHERE clause. So if we want to select, for example, the population from Germany, we would do it like this: SELECT population FROM world WHERE name = 'Germany';

We can select many columns from the table, we can do this by separating the name of the fields with a comma (,). In the WHERE clause we can also check items of a list, so to do that we use the word IN as follows: SELECT name, population FROM world WHERE name IN ('Brazil', 'Russia', 'India', 'China');

When it cames to numbers, we can select from between a range of them, that means, we can ask in SQL if there are some values that belongs to an specific range. We can do that with the BETWEEN clause and the word AND, the syntax to do that is like this: SELECT name, area FROM world WHERE area BETWEEN 250000 AND 300000;

There are many many more options to retrieve data from a database, to know more about it, you can check the syntax in the documentation of MySQL, that is an opensource database management system.

INSERT Basics

We have been retrieving data from the database, but we haven't seen how to insert actual data on it. You can do this by using the keywords INSERT INTO following by the name of the table, then use the keyword VALUE following by the values on parenthesis, in its most simple form, inserting a data looks like this: INSERT INTO countries VALUES ('my_country', 'America', 2381741, 37100000, 188681000000); When you use this form you have to follow the column order in the table definition, otherwise you will get an error.

So, if you want to insert the data in other order, or if you only have certain data to insert into the table, you can change the order on which you will insert the data. This can be done by putting the columns on parenthesis after the table name, using the same example as before, the syntax looks like follows: INSERT INTO countries (continent, name, area, population, gdp) VALUES ('America', 'my_country', 2381741, 37100000, 188681000000); Using the syntax that we have seen before it's possible to insert another set of data just by putting a comma between the parentheis of every set.

DELETE Basics

Deleting data from a database it's rather simple (that's the main reason why you should be careful about it). In its most simple form, you use the keywords DELETE FROM following by the name of the table to delete all rows in a table. DELETE FROM countries;

So, you can delete data given a condition, to do this, you can use the clause WHERE that we have seen before. So, an example of this would look as follows: DELETE FROM countries WHERE name = 'Mexico'; To learn more about this, you can check the MySQL Documentation.
